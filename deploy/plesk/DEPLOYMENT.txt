PLESK DEPLOYMENT PACKAGE

Layout:
- backend/: Node.js API app (set startup file to server.js)
- frontend/httpdocs/: static frontend files for admin domain
- uploads/: shared uploads mounted by backend at /uploads

Required steps in Plesk:
1) Backend domain (api.admin.hezzni.com)
   - Application root: backend
   - Startup file: server.js
   - Node.js: 18+
   - Run npm install in backend
   - Set backend environment variables (DB_*, JWT_SECRET, ADMIN_*)
   - Set UPLOADS_DIR=../uploads

2) Frontend domain (admin.hezzni.com)
   - Upload frontend/httpdocs contents into domain httpdocs
   - Ensure .htaccess exists for SPA routing

3) Shared uploads
   - Keep uploads folder beside backend
